⚙️ ตั้งค่าการเชื่อมต่อ Google Sheets

  1. ไปที่ Google Sheets → สร้าง Spreadsheet ใหม่
  2. เปิด Extensions → Apps Script → วางโค้ดจากไฟล์ Code.gs ที่แนบมา
  3. กด Deploy → New deployment → Web app
  4. ตั้ง Execute as: Me และ Who has access: Anyone แล้วกด Deploy
  5. คัดลอก Web app URL มาวางด้านล่าง แล้วกด บันทึก
